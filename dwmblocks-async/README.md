# dwmblocks-async
## Dependencies
- duf (for displaying disk stuff)

## Installation
It's pretty much the same as dwm but with one extra step:

```bash
$ cd dwmblocks-async/
$ sudo make clean install
$ cd modules/
$ ./install.sh
```

If you can't execute `install.sh`:

```bash
$ chmod +x ./install.sh
$ ./install.sh
```
